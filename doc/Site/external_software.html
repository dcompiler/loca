Required Software Packages

 

All the packages support Linux, Windows and Mach OS.  Follow the provide link and choose the version for the right operating system.


    Client:

  1. For binary instrumentation, install Pin from http://www.pintool.org/downloads.html .

  2. The analysis code is written in C for maximal performance.  It needs GCC (or LLVM on Mac).  The Pin tool also requires a compiler.

  3. The rest of the tool is written in Ruby and can run with either Ruby or JRuby.  To install Ruby on Mac, we recommend using Homebrew. For JRuby, go to http://jruby.org/download.  The last tested release, 9.1.17 (2.3.3), is fine.  The installation does not require a super user.  For a normal user install, see the post at http://key2start.wordpress.com/2011/05/03/install-jruby-in-linux-ubuntu-system/.  It is written for JRuby 1.6.0 (the tar command has a typo saying 1.3.1), which is also fine for our tool.  Make sure you can run gem install since we'll installing several Ruby tools.

  4. To report and view the result, you need to install two gems with ‘gem install launchy’ and ‘gem install rest-client’.

    5. Server:

  5. For visualization, it needs RScript.  To download,  go to http://www.r-project.org/ and click on CRAN. The instructions require superuser privilege (e.g. running sudo apt-get install package).

  6. To handle the request from client, the server need a gem called rack, install with ‘gem install rack’

  7. The plotted graphs are served through a web browser interface.  To enable it, we need a web browser.
