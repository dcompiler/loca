Locality Lab Installation

 

  1. Install Pin, GCC, Ruby (or JRuby), and RScript according to instructions on the following page.

  2. Unpack the directory.  Let [dir] be the root directory.  The client is in [dir]/gem/lib.  The server is in [dir]/server.

  3. To setup the environment, you need to run ‘[dir]/gem/lib/loca config’. It asks two questions: the first is the path of the Pin tool, e.g. /localdisk/cding/pin/pin-2.9-39599- gcc.3.4.6-ia32_intel64-linux/. The second is the host and path of the data repository. Now we use the same machine, simply use the default (by skipping the answer).

  4. To view the result locally, you need to install two gems with ‘gem install launchy’ and ‘gem install rest-client’.

  5. To test a program, run ‘[dir]/gem/lib/loca run’ followed by the program command and parameters, e.g. ‘[dir]/gem/lib/loca run gzip big_file’.

  6. To browse the result, first run ‘thin start -R [dir]/server/server.ru’. After execution, the result will be plotted on an open browser window.
